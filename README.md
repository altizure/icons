## Altizure Icons

### Install

```bash
yarn add @altizure/icons
```

### Usage
```javascript
import { AltizureLogo } from '@altizure/icons'

class MyClass extends React.Component {
  render() {
    return <AltizureLogo />
  }
}
```

Icons can be configured with inline props:
```javascript
<AltizureLogo size={64} round />
```
