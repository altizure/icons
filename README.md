## Altizure Icons

### Install

```bash
yarn add @altizure/icons
```

### Usage
``` js
import { Altizure } from '@altizure/icons'

class MyClass extends React.Component {
  render() {
    return <Altizure />
  }
}
```

Icons can be configured with inline props:
```javascript
<Altizure size={64} round />
```
